Perguntas sobre HTML

João Ramos

1. O que é HTML? E para que serve?

1. HTML ou HyperText Markup Language é uma linguagem de marcação.
O HTML é um componente base da WEB, com isto quero dizer que o HTML permite a contrução de websites, com HTML podemos também adicionar imagens e vídeos, por meio de Hipertextos.

2. Que ferramentas são necessárias para construir um site em HTML?

2. Um progama que nos permita escrever código (vscode, virtual studio, até mesmo o notepad).

3.Quais os elementos base necessários para a criação de uma página web em HTML?

Os elementos base são: <!DOCTYPE>, <html>, <head> <body>.

4. O que é um elemento? Dê exemplos.

4. Um elemento é tudo o que está dentro de uma start tag até à end tag, como por exemplo:
<p>Este é um parágrafo de teste </p>.

5. O elemento <br> necessita de ser fechado? Justifique.

5. Não, o elemento <br> é uma empty tag, empty tags não têm end tags, logo não podem ser fechados.

6.Para que servem os atributos? Dê exemplos.

6. Os atributos servem para dar informação adicional sobre os elementos.
Um exemplo disto é o atributo "tag", que serve para especificar o URL do ficheiro media que pretendemos colocar no nosso site.
